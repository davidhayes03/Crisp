Demo Features

This demo showcases a hero section using static images with animation effects instead of video.

  • 2 Stock Photos - High-quality Unsplash images cycling every 8 seconds
  • Ken Burns Effect - Slow zoom and pan animation on each image (20s loop)
  • Crossfade Transition - Smooth 2-second fade between images
  • Animated Logo - SVG stroke animation drawing the logo (3s)
  • Staggered Text - "For the creatively curious" slides up from bottom right
  • Content Box - Slides in from left with description and CTA

How to use with your own content:

1. Replace the Unsplash URLs with your own high-res property photos (1920x1080 minimum)

2. Upload your Crisp logo SVG to the images folder and reference it

3. Adjust animation timings in the CSS (search for "animation:" properties)

4. Update text content to match your messaging
